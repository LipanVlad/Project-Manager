# Project-Manager
A full stack project management application built with Spring Boot, Spring Security with JWT Authentication, and a vanilla JavaScript frontend. It allows users to register, log in, and manage their projects, tasks, and subtasks.

## Features
- Register account
- Login
- Automatically loaded projects based on user_id
- Create projects
- Add tasks to projects
- Add subtasks to tasks

## Technologies used
- Spring Boot 3.x
- Spring Security 6.x
- JJWT
- PostgreSQL
- Spring Data JPA with Hibernate
- Maven
- Javascript with CSS and HTML
